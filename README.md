Network-Traffic-Security-Dashboard

This project analyzes network traffic logs and detects suspicious activities using Python.
It performs data cleaning, traffic analysis, security monitoring, and visualization.

Features
- Data cleaning
- Traffic analysis
- Top IP detection
- Failed request monitoring
- Suspicious activity detection
- Visualization dashboard

  Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

  Dataset
   Dataset Source
   https://share.google/uVvrXX2PjIUDpbk3s

  Workflow
1. Data Loading
2. Data Cleaning
3. Traffic Analysis
4. Suspicious Activity Detection
5. Visualization
6. Security Insights Generation

    Security Logic
- Detect high traffic from single IP
- Detect excessive failed requests
- Detect suspicious URL access
- Detect abnormal request spikes

  Networking Concepts
- IP Addresses
- HTTP Status Codes
- Network Traffic Patterns
